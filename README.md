# Yuan Lei Angelo J. Guan | Personal Portfolio

Welcome to the repository for my digital portfolio! 

I am a Computer Engineering Technology student and Multimedia Artist. This website serves as a curated showcase of my work, bridging the gap between "under-the-hood" technical infrastructure and "front-facing" creative design. 

## ✨ Key Features
This portfolio was built from scratch focusing on clean code, smooth animations, and a highly interactive user experience:
* **Interactive Media Carousel:** A custom JavaScript-powered rotating gallery that seamlessly opens full-screen video and image modals.
* **Bento Box UI:** A modern, fully responsive CSS Grid layout used to elegantly display photography and multimedia assets.
* **Scroll-Triggered Animations:** Utilizes the IntersectionObserver API to create smooth, cascading text reveals and a dynamic side-scroll navigation wheel.
* **Responsive Design:** Carefully crafted media queries ensure the layout gracefully adapts from ultra-wide desktop monitors down to mobile screens, complete with a custom slide-out hamburger menu.

## 🎬 Projects Featured
This portfolio highlights my work across various disciplines, including:
* **Short Films & Video Production:** Highlighting projects like *FLOWERS*, *RIZAL*, and *HARANA* with a focus on color grading, pacing, and visual storytelling.
* **Technical Apparel Design:** 3D digital mockups and branding for projects like the CETE Jersey and YUNKY streetwear.
* **Graphic Design & Photography:** Concept posters, UI/UX interfaces, and a diverse photography gallery.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure and custom data attributes.
* **CSS3:** Advanced Grid/Flexbox layouts, fluid typography, and CSS transitions.
* **Vanilla JavaScript:** DOM manipulation, custom event listeners, and modular interactivity without relying on heavy external libraries.
